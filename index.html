🚀 快速开始

只需几步即可开始使用 GeBuDiu API 进行文档翻译。以下是最简单的入门指南。

检查服务状态

首先确认 API 服务运行正常

bash 
curl https://gebudiu-api.onrender.com/health
翻译文本

发送第一个翻译请求

bash 
curl -X POST https://gebudiu-api.onrender.com/api/v1/translate \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, world!",
    "source_lang": "en",
    "target_lang": "zh"
  }'
翻译 DOCX 文档

上传并翻译 Word 文档

bash 
curl -X POST https://gebudiu-api.onrender.com/api/v1/translate/docx \
  -F "file=@document.docx" \
  -F "source_lang=en" \
  -F "target_lang=zh"

GeBuDiu API 提供 RESTful 接口，所有请求和响应均使用 JSON 格式。

文本翻译

POST /api/v1/translate

翻译文本内容，支持翻译记忆库和异步模式

请求参数

参数 类型 必填 说明
text string 是 待翻译文本
source_lang string 否 源语言代码 (默认: auto)
target_lang string 是 目标语言代码
use_tm boolean 否 使用翻译记忆库 (默认: true)
async boolean 否 异步模式 (默认: false)

语言代码

代码 语言
zh中文
en英文
ja日文
ko韩文
auto自动检测

响应示例

json 200 OK 
{
  "translation": "你好，世界！",
  "source_lang": "en",
  "target_lang": "zh",
  "source": "api",
  "confidence": 0.95,
  "processing_time": 0.823
}

DOCX 文档翻译

POST /api/v1/translate/docx

上传并翻译 Word 文档 (.docx)，保留原文档格式

请求参数

参数 类型 必填 说明
file file 是 DOCX 文件 (最大 10MB)
source_lang string 是 源语言代码
target_lang string 是 目标语言代码
preserve_format boolean 否 保留格式 (默认: true)
domain string 否 指定领域 (覆盖自动检测)

💻 代码示例

使用 Python、JavaScript 或 cURL 快速集成 GeBuDiu API。

Python 示例

python 
import requests

API_BASE = "https://gebudiu-api.onrender.com"

def translate_text(text, source_lang="en", target_lang="zh"):
    """翻译文本"""
    response = requests.post(
        f"{API_BASE}/api/v1/translate",
        json={
            "text": text,
            "source_lang": source_lang,
            "target_lang": target_lang
        }
    )
    return response.json()

# 使用示例
result = translate_text("GeBuDiu API makes translation easy.")
print(result["translation"])
# 输出: GeBuDiu API 让翻译变得简单。

JavaScript 示例

javascript 
const API_BASE = 'https://gebudiu-api.onrender.com';

async function translateText(text, sourceLang = 'en', targetLang = 'zh') {
  const response = await fetch(`${API_BASE}/api/v1/translate`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      text,
      source_lang: sourceLang,
      target_lang: targetLang
    })
  });
  return response.json();
}

// 使用示例
translateText('Hello, world!')
  .then(result => console.log(result.translation));

⚠️ 错误处理

了解 API 错误码和相应的处理方案。

HTTP 状态 错误码 说明 解决方案
400 INVALID_REQUEST 请求格式错误 检查 JSON 格式
400 MISSING_FIELD 缺少必填字段 检查 text 和 target_lang
400 INVALID_LANGUAGE 不支持的语言 使用支持的语言代码
400 FILE_TOO_LARGE 文件过大 文件大小限制 10MB
429 RATE_LIMITED 请求过于频繁 降低请求频率
500 TRANSLATION_ERROR 翻译服务错误 稍后重试
503 SERVICE_UNAVAILABLE 服务暂时不可用 检查 /health 端点

💡 最佳实践

翻译优化建议

✅
推荐做法
  • 批量短文本优先使用同步模式
  • 长文档 (>5000 字) 使用异步模式
  • 重复内容多的文档开启 TM
  • 专业文档指定 domain 参数
❌
避免做法
  • 单请求翻译过长文本 (>10000 字)
  • 频繁请求相同内容 (使用 TM)
  • 忽略领域检测 (影响专业术语)
